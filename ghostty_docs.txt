# Ghostty Configuration Documentation

This file contains documentation for all Ghostty configuration options.

## Appearance

### font-family

The font family to use for the terminal. This can be a single font family name or a comma-separated list of font families.

Type: string
Default: monospace
Example: font-family = JetBrains Mono

### font-size

The font size in points.

Type: number
Default: 12
Min: 6
Max: 72
Example: font-size = 14

### theme

The color theme to use. Ghostty comes with several built-in themes.

Type: enum
Default: default
Options: default, nord, dracula, solarized-dark, solarized-light, monokai, gruvbox-dark, gruvbox-light
Example: theme = nord

### cursor-style

The style of the cursor.

Type: enum
Default: block
Options: block, bar, underline
Example: cursor-style = bar

### cursor-blink

Whether the cursor should blink.

Type: boolean
Default: false
Example: cursor-blink = true

### background-opacity

The opacity of the terminal background. 1.0 is fully opaque, 0.0 is fully transparent.

Type: number
Default: 1.0
Min: 0.0
Max: 1.0
Example: background-opacity = 0.95

### background-blur-radius

The blur radius for the background when using transparency. Only works on supported platforms.

Type: number
Default: 0
Min: 0
Max: 100
Example: background-blur-radius = 20

## Window

### window-width

The initial window width in pixels.

Type: number
Default: 800
Min: 400
Max: 7680
Example: window-width = 1200

### window-height

The initial window height in pixels.

Type: number
Default: 600
Min: 300
Max: 4320
Example: window-height = 800

### window-padding-x

Horizontal padding inside the window in pixels.

Type: number
Default: 2
Min: 0
Max: 100
Example: window-padding-x = 10

### window-padding-y

Vertical padding inside the window in pixels.

Type: number
Default: 2
Min: 0
Max: 100
Example: window-padding-y = 10

### window-title

The initial window title.

Type: string
Default: Ghostty
Example: window-title = My Terminal

## Terminal

### shell

The shell to use. If not specified, Ghostty will use the user's default shell.

Type: string
Default:
Example: shell = /bin/zsh

### scrollback-lines

The number of lines to keep in the scrollback buffer.

Type: number
Default: 10000
Min: 0
Max: 100000
Example: scrollback-lines = 50000

### mouse-hide-while-typing

Hide the mouse cursor while typing.

Type: boolean
Default: true
Example: mouse-hide-while-typing = false

### copy-on-select

Automatically copy selected text to clipboard.

Type: boolean
Default: false
Example: copy-on-select = true

## Input

### keybind

Define a keybinding. This property can be specified multiple times. Format: keybind = key>action

Type: repeatable
Default:
Example: keybind = ctrl+shift+c>copy_to_clipboard

### mouse-scroll-multiplier

Multiplier for mouse scroll speed.

Type: number
Default: 1.0
Min: 0.1
Max: 10.0
Example: mouse-scroll-multiplier = 1.5

## Platform-Specific

### macos-titlebar-style

The style of the titlebar on macOS.

Type: enum
Default: native
Options: native, transparent, hidden
Platform: macOS
Example: macos-titlebar-style = transparent

### macos-option-as-alt

Treat the Option key as Alt on macOS.

Type: enum
Default: false
Options: true, false, left, right
Platform: macOS
Example: macos-option-as-alt = true

### linux-csd

Enable client-side decorations on Linux.

Type: boolean
Default: true
Platform: Linux
Example: linux-csd = false

### gtk-titlebar

Use GTK titlebar on Linux with GTK.

Type: boolean
Default: true
Platform: Linux (GTK)
Example: gtk-titlebar = false

## Color Palette

### palette

Define a color palette entry. This property can be specified multiple times. Format: palette = index=#hexcolor

Type: repeatable
Default: (ANSI standard colors)
Example: palette = 0=#1d1f21

## System

### working-directory

The initial working directory for new shells.

Type: string
Default: (home directory)
Example: working-directory = /Users/myuser/projects

### auto-update

Enable automatic updates.

Type: enum
Default: check
Options: off, check, download
Example: auto-update = download

## Notifications

### bell-action

Action to take when the bell character is received.

Type: enum
Default: visual
Options: none, visual, sound, system
Example: bell-action = system

### desktop-notifications

Enable desktop notifications for terminal bell.

Type: boolean
Default: true
Example: desktop-notifications = false

## Config

### config-file

Include another config file. This property can be specified multiple times.

Type: repeatable
Default:
Example: config-file = ~/.config/ghostty/theme.conf
